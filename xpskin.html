How to install:
Step 1. Install the file linked below and make sure to install to C:\WinXP
Step 2. Once the setup has finished the Classic Shell setup should open and install it
Step 3. Right-Click on your start button and click settings, then click Backup > Restore from XML
Step 4. Go to C:\WinXP\MenuType and choose whether you want your regular start menu or Windows XP one
Step 5. Open the respective XML file in Classic Shell
Step 6. Choose your background from C:\WinXP\bg
